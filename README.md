# Front-End

Repositório com arquivos estáticos (HTML/CSS/JS) para o projeto Front-End.

## Estrutura

- `index.html` — página principal
- `tabela.html` — tabela de exemplo
- `TesteKamylla.html` — arquivo de teste
- `PROJETO/` — arquivos do projeto (ex.: `MENU/`)

## Como visualizar localmente

Opções rápidas:

1. Usando a extensão Live Server (recomendado para desenvolvimento)
	- Abra o VS Code no workspace.
	- Abra um arquivo HTML (ex.: `index.html`).
	- Clique em "Go Live" na barra de status ou clique com o botão direito e escolha "Open with Live Server".
	- Por padrão o workspace está configurado para usar porta `5500` e navegador `chrome` (veja `/.vscode/settings.json`).

2. Usando `live-server` via npm:
	```bash
	npm install -g live-server
	live-server .
	```
